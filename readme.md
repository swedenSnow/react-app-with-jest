# React App with Jest

## Requires

- [Node](https://nodejs.org/en/download)
- [npm](https://www.npmjs.com/)

### Install

```bash
$ npm
```

## Development

```bash
$npm run dev-server # will start the dev server on port 8080
```

```bash
$npm test -- --watch # will run all test suites with watch flag
```
